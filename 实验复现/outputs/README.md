# 输出目录说明

- `outputs/demo/`: 轻量 demo 运行结果。
- `outputs/repro_run/`: 使用真实输入时建议输出目录。

示例输出包括：

- `metrics/rf_metrics.json`
- `metrics/baseline_metrics.json`
- `metrics/predictions_demo.csv`
- `metrics/rf_selection_scores.csv`
- `feature_importance/feature_importance_demo.csv`
- `feature_importance/representative_features_demo.csv`
- `figures/prediction_scatter_demo.png`
- `figures/feature_importance_demo.png`

注：这些输出用于流程验证，非论文正式数值复现结果。
