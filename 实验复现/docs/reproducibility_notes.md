# 可复现性说明

## 已固定设置

- 随机种子：`42`
- 训练测试划分：`7:3`
- 交叉验证：`5-fold`
- Bootstrap：训练集内扩增到 `480`
- 共线性阈值：Spearman `|r| > 0.7`

## 防数据泄漏策略

- 先划分训练/测试集。
- 再仅在训练集拟合预处理（标准化）。
- 测试集仅使用训练集统计量做变换。

## 模型选择策略

- RF 综合分数 = `mean(CV R², OOB R²)`。
- 论文最优参数保留为默认：`500/10/4`。
- 完整网格范围接口保留在配置文件中，可手动启用。

## 轻量运行边界

- 默认不执行长时间全网格搜索和大规模重训练。
- 富集分析仅保留接口与说明，不在默认流程中运行。
