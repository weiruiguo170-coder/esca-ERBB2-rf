# 项目概览

本目录用于提供论文对应的轻量复现流程，而非一次性跑完所有重计算实验。

## 目标

- 让使用者在几分钟内跑通完整复现流程。
- 保留正式复现接口（参数搜索、真实数据替换、特征解释）。
- 与投稿版论文方法保持一致的关键设置。

## 当前主流程

1. 读取整理后的 GDSC+CCLE 输入 CSV
2. 构建两药联合 `log2(IC50)` 终点
3. 训练/测试 `7:3`（种子 42）
4. 仅训练集拟合预处理
5. 训练集 Bootstrap 到 480
6. RF 训练与模型选择（CV R² + OOB R²）
7. LR/SVR 对照
8. 杂质+置换重要性 + Spearman 共线性簇代表特征
