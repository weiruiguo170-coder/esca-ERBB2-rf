# Data Guide

This repository does not include original research data, patient-level data, or downloaded matrices.

Users must download public data from GDSC and CCLE and prepare local input files in `data/input/`.

What this repository provides:
- Field templates in `data/schema/`
- Data validation and preprocessing scripts in `scripts/`
- End-to-end reproducibility workflow aligned with the manuscript methodology

What this repository does not provide:
- Raw GDSC/CCLE files
- Any identifiable personal or institutional information
- Intermediate heavy outputs

Compliance note:
Users are responsible for following all terms, licenses, and usage policies of external databases (including GDSC/CCLE).
